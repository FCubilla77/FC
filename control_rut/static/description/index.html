Localización fiscal para Paraguay — validación y renombrado de RUT

Este módulo adapta los formularios de contactos y compañías de Odoo al contexto fiscal paraguayo, renombrando el campo estándar de identificación fiscal como RUT y validando su dígito verificador según el algoritmo oficial.

✔

Validación automática

Verifica el dígito verificador del RUT al crear o modificar un contacto.

🏷

Etiquetas adaptadas

Cambia las etiquetas de "NIF/RUT" a "RUT" en los formularios estándar.

⚙

Opción de omisión

Permite omitir la validación para casos particulares desde el propio contacto.

Funcionalidades

  • Limpieza automática de caracteres no numéricos en el RUT
  • Validación del dígito verificador (módulo 11) al crear o guardar
  • Campo "Omitir validación de RUT" para excepciones puntuales
  • Campo de solo lectura para contactos con empresa asociada

Compatibilidad
